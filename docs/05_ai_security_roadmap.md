# 5. AI & Security Roadmap

## 5.1 AI Agent Architecture
- **Commodity Forecasting**: Time-series ML models predict price trends.
- **Supplier Risk**: NLP models scan news feeds to adjust geopolitical and financial risk scores dynamically.
- **Procurement Copilot**: LLM interface for natural language querying of spend data.

## 5.2 Security Architecture
- **Authentication**: JWT with short expiries and refresh tokens.
- **Authorization**: Strict RBAC middleware.
- **Data Protection**: AES-256 encryption for sensitive supplier banking details.
- **API Security**: Rate limiting, Helmet.js for headers, CORS policies.

## 5.3 Future Roadmap
- Phase 2: Blockchain-based immutable audit logs for public sector compliance.
- Phase 3: IoT integration for supply chain tracking directly linking to Smart Contracts.
